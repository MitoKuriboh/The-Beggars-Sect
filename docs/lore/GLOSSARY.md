# PRONUNCIATION GUIDE

## Consonants

| Letter | Sound | Example |
|--------|-------|---------|
| Q | "ch" | Qi = "chee" |
| X | "sh" | Xiu = "shyo" |
| Zh | "j" | Zhong = "jong" |
| C | "ts" | Cai = "tsai" |
| R | between r/j | Ren = "ren" |

## Vowels

| Letter | Sound | Example |
|--------|-------|---------|
| A | "ah" | Da = "dah" |
| E | "uh" | De = "duh" |
| I | "ee" | Li = "lee" |
| O | "oh" | Bo = "boh" |
| U | "oo" | Wu = "woo" |
| Ü | "ü" (German) | Nü = "nü" |

## Tones (Simplified)

For gameplay purposes, tones are indicated but not critical:
- First tone (ā): high, flat
- Second tone (á): rising
- Third tone (ǎ): dipping
- Fourth tone (à): falling
